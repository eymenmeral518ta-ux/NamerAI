## üöÄ Run and Deploy

This section details how to install the app on your local machine and access your project in AI Studio.

### üõ†Ô∏è Prerequisites

To run the application locally, you must have **Node.js** installed on your system.

### üíª Local Installation Steps

1.  **Install Dependencies:**
    Open the terminal in the project directory (where this README is located) and install all the required packages:

    ```bash
    npm install
    ```

2.  **Set the API Key:**
    * Open the **`.env.local`** file located in the project root directory.
    * Fill in the **`GEMINI_API_KEY`** variable with your Gemini API key obtained from Google AI Studio:

    ```bash
    # .env.local
    GEMINI_API_KEY="AIzaSy...Your Gemini API Key Goes Here...XYZ"
    ```

3.  **Start the Application:**
    Start the development server. The application will typically start at **http://localhost:3000**.

    ```bash
    npm run dev
    ```
